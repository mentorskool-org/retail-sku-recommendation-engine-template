# Scenario 02 — Understanding CPG Distribution (Hands-On)

**Stack:** Excel  •  **Duration:** ~2h 30m  •  **Maps to sub-scenarios:** 1, 2, 3

## Overview

Apply your understanding of CPG distribution by mapping the journey of a Dove shampoo bottle from factory to consumer, analysing a real sales representative's daily report, and calculating distribution metrics for a set of retail outlets. This scenario bridges theoretical knowledge with practical retail operations.

## Pre-requisites

- Completed: CPG Distribution & Retail Operations masterclass (Position 1)
- Excel: basic formulas, charting

## Learning Objectives

1. Create a visual flowchart showing the complete distribution journey with margin analysis at each node.
2. Calculate productive call rate, average order value, and OOS rate from sales data.
3. Analyse the relationship between SKU count and sales performance across stores.
4. Profile store characteristics and recommend visit frequencies based on sales potential.

## Deliverables (commit into this folder)

- `distribution-journey.xlsx` — flowchart + margin analysis at each distribution node.
- `sales-rep-metrics.xlsx` — productive call rate, AOV, OOS calculations from the daily report.
- `store-profiling.xlsx` — SKU-count vs sales analysis and visit-frequency recommendations.
- `notes.md` — short write-up of insights and any assumptions.
