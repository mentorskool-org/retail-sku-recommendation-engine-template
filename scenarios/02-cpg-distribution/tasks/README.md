**Total hands-on time:** ~2h 30m

## How to use the starter workbooks

Tasks 2 and 3 ship with **pre-built Excel workbooks** that contain the raw data + exercise templates with yellow-highlighted cells. Your workflow:

1. Open the starter workbook (e.g., `task-02-sales-rep-analysis-workbook.xlsx`).
2. Work through each sheet (`Ex1`, `Ex2`, ...) filling the yellow cells with your calculations. Formula hints are on every sheet.
3. **Save As** the completed workbook to the **scenario root** with the deliverable name:
   - Task 2 → save as `../sales-rep-metrics.xlsx`
   - Task 3 → save as `../store-profiling.xlsx`
4. Leave the original starter workbooks in this folder unchanged — they're reference copies.

Task 1 has no starter workbook — build `distribution-journey.xlsx` from scratch using the reference data tables in the task brief.
